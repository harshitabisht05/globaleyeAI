## 📦 Installation

1. **Clone the repository**  
   ```bash
   git clone https://github.com/harshitabisht05/globaleyeAI.git
   cd globaleye-ai
    ```
2. **Install dependencies**
    ```bash
    npm install
    ```
3. **Run the development server**
    ```bash
    npm run dev
    ```
4. **Open in browser**
- Navigate to http://localhost:3000
